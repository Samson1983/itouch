# 锁屏对对碰_锁屏应用

#### Description
拖动两小球碰撞在一起即可解锁屏幕。你可以从相册中选择你自己的图片， 

功能：

  1. 锁屏\解锁：拖动水晶球相撞即可解锁。

  2. 水晶球上的头像可以随意替换，可视化操作非常简易。

  3. 声音、振动效果自由设置。
